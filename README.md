# Verifiable and Multi-Keyword Searchable Attribute-Based Encryption

This project implements a secure cloud storage system that supports multi-keyword searchable encrypted data.

Features:
- Attribute-Based Encryption
- Multi-keyword search
- Secure cloud storage
- TF-IDF based ranking

Technologies Used:
- Java
- JSP
- JDBC
- MySQL
- Apache Tomcat
